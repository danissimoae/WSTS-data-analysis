# WSTS-data-analysis

## Работа посвящена определению типа процесса временного ряда, поиску значительных отклонений, выявлению сезонной компоненты, построению модели ARIMA, ARCH, GARCH и прогнозированию по временному ряду

### Итоговая цепочка лабораторных работ:

- **Лабораторная № 3** — удаление детерминированной сезонности методом
  4 гармонических пар;
- **Лабораторная № 4** — моделирование условной средней с помощью **ARMA()**;
- **Лабораторная № 5** — моделирование условной дисперсии остатков через
  **GARCH(1,2)** с t-распределением;
- **Лабораторная № 6** — построение и сравнение прогнозов; показано, что на
  коротком горизонте основное качество точечного прогноза определяется
  уравнением среднего, а GARCH полезен прежде всего для корректной оценки
  неопределённости, но не улучшает точечный прогноз.
